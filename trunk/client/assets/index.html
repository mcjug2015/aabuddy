Meeting Finder Overview

This app will help you to find meetings, add new meetings and correct information for existing meetings.

The main features are:
  • Find Meeting:
    • Select Meetings that interest you
    • Map and Directions get map and directions to an individual selected meeting
    • Mark Meeting Not There when meeting is not there or to correct meeting location or time.
  • Add Meeting: Create a record for a new meeting. You must be logged in to access this feature.

The addtition features are: Settings, and Log In / Log Out

Feature Details

  • Add Meeting
    Prerequisites: To add meeting you must first register with an email account and subsequently log in.

    Steps to create a meeting:
    1. Click the Add Meeting icon on the Main Screen.
    2. If the Login screen shows up (see a tip below), enter credential information.
    3. Enter Meeting Name, Start time, End time, Day of week, Address, and Meeting Description (an optional field). Also see a tip for End time below.
    4. Click the Validate Address button to check the address.
    5. Click the Submit button if all inputs are valid.

    Tips:
    • The Login screen will show up at step 2 if you have not logged in using the most current version of Meeting Finder.
    • Set the Start time FIRST and let the End time is set at one hour after the Start time.

    Hide Add Meeting content.

  • Find Meeting
    Steps to find meetings
    1. Click the Find Meeting on the Main Screen.
    2. Create a query by entering Meeting Name, Start time, End time, Days of week, Address, and Within. Also see tips for Start time and End time below
    3. Click the Search button to get the list of Meeting Search Results.

    Map of a meeting location
    • Long-click on an item in the list of Meeting Search Results to display a map.

    Mark Meeting Not There
    • If you go to a meeting location and the meeting is not there, please find the meeting and click the Not There button to warn others.
    • If you go to a meeting location and the meeting specifics are incorrect, please find the meeting, click the Not There button and provide correct information in the text box.

    Tips
    • Start time and End time: click the X icon to erase time and then find all meetings in a specific day

    Hide Find Meeting content.

  • Settings
    You may Change Password or Reset Password from this screen.
    You must be logged in to change your password. For Reset Password, see Log In / Log Out, Register and Reset Password for details.

    Hide Settings content.

  • Log In / Log Out, Register, and Reset Password
    Click the Log In icon to find three features, Login to Add a Meeting, Register, and Reset Password in the Log In screen. Note, the Log In icon will be replaced by the Log Out icon after you log in at the first time.

    Feature Details

    Login to Add a Meeting
    Using the same device you only have to log in once per version of Meeting Finder. There are two ways to log in
    1. Click the Add Meeting icon on the Main screen. The Add Meeting screen will show up after login or
    2. Click the Log In icon on the Main screen. The Main screen will show up after login. Note, the Log In icon in the Main screen is replaced by the Log Out icon after login.
    3. Enter the email address you use to register and your password
    4. Click Log In button

    Hide Login to Add a Meeting content.


    Log Out

    When the Log Out icon is shown in the Main screen, your credential information currently stored in your cell phone from your initial server logon. You do not need to re-log in again. Click the Log Out icon if you want
    • To Register with another account.
    • To Add Meeting with another account.
    • To Reset Password.

    Hide Log Out content.


    Register
    Before you can Add Meeting, you have to register. Steps to register:
    1. Click the Log In icon on the Main screen.
    2. Click the Register Now button to register.
    3. Enter all fields and click the Register Now button. Then, click OK.
    4. Meeting Finder will send a confirmation email to you and you have to click the link in the email to complete the registration.

    Hide Register content.


    Reset Password
    You have to Log Out on the Main screen if you have logged in to the server. Steps to reset password:
    1. Click the Log In icon on the Main screen.
    2. Click the Reset Password button in the Login screen.
    3. Meeting Finder will send a confirmation email to you and you have to click the link in the email.
    4. After you click the link, a reset password form will be displayed in a browser.
    5. Enter Username (i.e., your email address), New Password, and Confirm Password and then click the Submit button.

    Hide Reset Password content.
