AA Buddy Overview

This app will help you to post meetings and to find meetings you want to attend to.

The primary features are
  • Add Meeting: Create a meeting. You have to log in after you install a new version in your device.
  • Find Meeting:
    • Find Meetings you are interested in.
    • Show The Map for a meeting using a long click.
    • Mark Meeting Not There when a meeting is NOT held on time.
In the Main screen, there are three other features: Settings, Log In / Log Out, and Help.

Feature Details

  • Add Meeting See more
    Prerequisites: To add meeting is prohibited until you register with an email account and log in to.

    Steps to create a meeting:
    1. Click the Add Meeting icon on the Main Screen.
    2. (Optional) If the Login screen shows up (see a tip below), enter credential information.
    3. Enter Meeting Name, Start time, End time, Day of week, Address, and Meeting Description (an optional field). Also see a tip for End time below.
    4. Click the Validate Address button to check the address.
    5. Click the Submit button if all inputs are valid.

    Tips:
    • The Login screen will show up at step 2 if you have not logged in using the existing version of AA Buddy.
    • Set the Start time FIRST and let the End time is set at one hour after the Start time.

    Hide Add Meeting content.

  • Find Meeting See more
    Steps to find meetings
    1. Click the Find Meeting on the Main Screen.
    2. Create a query by entering Meeting Name, Start time, End time, Days of week, Address, and Within. Also see tips for Start time and End time below
    3. Click the Search button to get the list of Meeting Search Results.

    Map of a meeting location
    • Long-click on an item in the list of Meeting Search Results to display a map.

    Mark Meeting Not There
    • When you go to a meeting location, the meeting is not there. Please find the meeting and click the Not There button to notice other persons.

    Tips
    • Start time and End time: click the X icon to erase time and then find all meetings in a specific day

    Hide Find Meeting content.

  • Settings See more
    There are three settings: Meeting Server Secure Base URL, Meeting Server Unsecure Base URL, and All certificates will be trusted. They will be used for future versions. Please do not make changes in the current version.
    You may Reset Password from this screen.

    Hide Settings content.

  • Log In / Log Out, Register, and Reset Password See more
    Click the Log In icon to find three features, Login to Add a Meeting, Register, and Reset Password in the Log In screen. However, the Log In icon will be replaced by the Log Out icon after you log in at the first time. See those features in details below.

    Login to Add a Meeting More
    You need to log in to the same device only one time with the existing version of AA Buddy. There are two ways to log in to
    1. Click the Add Meeting icon on the Main screen. The Add Meeting screen will show up after login.
    2. Click the Log In icon on the Main screen. The Main screen will show up after login.
    The Log In icon in the Main screen is replaced by the Log Out icon after login.

    Hide Login to Add a Meeting content.


    Log Out More
    When the Log Out icon is shown in the Main screen, it means that your credential information has stored in your cell phone when you logged in to the server. You do not need to re-log in again. You want to click the Log Out icon if you want
    • To Register with another account.
    • To Add Meeting with another account.
    • To Reset Password.

    Hide Log Out content.


    Register More
    Before you want to Add Meeting, you have to register. Steps to register:
    1. Click the Log In icon on the Main screen.
    2. Click the Register Now button to register.
    3. Enter all fields and click the Register Now button. Then, click OK.
    4. AA Buddy will send a confirmation email to you and you have to click the link in the email to complete the registration.

    Hide Register content.


    Reset Password More
    You have to Log Out on the Main screen if you have logged in to the server. Steps to reset password:
    1. Click the Log In icon on the Main screen.
    2. Click the Reset Password button in the Login screen.
    3. AA Buddy will send a confirmation email to you and you have to click the link in the email.
    4. After you click the link, a reset password form will be displayed in a browser.
    5. Enter Username (i.e., your email address), New Password, and Confirm Password and then click the Submit button.

    Hide Reset Password content.
